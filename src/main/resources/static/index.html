接口说明

  • GET /api/chat/health 用于检查 demo 是否启动成功。
  • GET /api/chat/simple?message=... 用于快速试跑。
  • POST /api/chat 用于标准 JSON 调用，Body 为 {"message":"..."}。

启动前准备

  • 先设置环境变量 AI_DASHSCOPE_API_KEY。
  • 默认模型是 qwen-plus，也可以通过 DASHSCOPE_MODEL 覆盖。
  • 如果回答报错，先访问 /api/chat/health 确认服务已启动，再检查 Key 和网络。
